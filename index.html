ssh 反向隧道连接学校内部服务器

1 介绍

通常访问内网的服务器，只需打开服路由器的端口映射就行了，但是在学校的网关对访问的 ssh 进行了限制，不能访问。因此本文使用 ssh 建立反向隧道，曲线救国。好处是可以访问并 且不用进行端口映射，缺点也很明显需要一台公网主机。

2 准备

公网主机（腾讯云）一台 A，内网主机 B(能够连接外网)，外网主机 C

就是说在内网 B 连接 A 的时候，实际上已经建立了 A->B 的连接。把 A 的端口映射到 B 的端 口上，通过 C 访问被映射 A 的端口，进而访问 B。

注：腾讯云学生一个月 1 元，还是非常厚道的。

[转]How to learn Machine learning

1 Read the books

  • Kevin Murphy的机器学习
  • Bishop 的 PRML
  • Daphne Koller的概率图模型（Probabilistic Graphical Models: principles and Techniques）
  • David Mackay的信息理论：推理与学习方法

Key: 不要想着一开始就能获得某个知识的所有方面，尽量从多个不同的资源去阅读对同一 个知识的描述。

2 Read the paper

  • 选择一篇你喜欢的论文，然后花一个星期深入研究它，花时间去琢磨每一个细节，没有变 量，把推导的省略部分补全。
